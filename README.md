# Qt-Examples

## 1. Branch Namimg Specification

* **branch namimg format**: (product name)\_(software name)\_(Qt version)
  * **product name**:
    * SBC-7112S;
    * SBC-7112S-Q;
    * SBC-7109S-454;
  * **Qt version**:
    * Qt4.8.5;
    * Qt5.5;
* **E.g:**  
`SBC-7112S`(product name)`_UARTTest`(software name)`_Qt4.8.5`(linux version) = `SBC-7112S_UARTTest_Qt4.8.5`
