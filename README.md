## 二、原理

### 2.1 背光调节

* 背光设备节点：`/sys/class/backlight/pwm-backlight.1/brightness`。
* 背光值范围`0~248`，这里是字符串。
* 将`0~248`范围内的字符串写入上面的背光设备节点内，就会设置背光亮度。
